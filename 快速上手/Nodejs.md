# npm vs yarn

* 在npm5.0之前，yarn的优势特别明显
* yarn优点：并行安装、离线模式、安装版本统一、简洁输出、多注册来源处理、更好的语义化 https://zhuanlan.zhihu.com/p/27449990

```bash
# i = install
npm i === yarn
npm i taco === yarn add taco
# un = uninstall
npm un taco === yarn remove taco
# dev version
npm i -D taco === yarn add taco --dev
# up = update
npm up === yarn upgrade
```
