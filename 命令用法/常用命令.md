# basename

## 基本文件名

```bash
basename /usr/bin/file
file
```

## 去除文件后缀

```bash
basename /usr/include/stdio.h .h
stdio

basename -s .h /usr/include/stdio.h
stdio
```

## 批量文件名

```bash
# 以\n分隔
basename -a dir1/file1 dir2/file2
file1
file2

# 合并一起
basename -a -z dir1/file1 dir2/file2
file1file2
```

# dd

## 删除开头部分数据

```bash
# 删除开头 4字节*53 = 212字节
dd if=input.ts of=output.ts bs=4 skip=53
```

# grep

## -E 匹配多个关键字合并输出

```bash
cat /proc/cpuinfo | grep -E 'processor|model name|cpu MHz'
processor	: 0
model name	: Intel(R) Xeon(R) Platinum 8259CL CPU @ 2.50GHz
cpu MHz		: 2499.998
processor	: 1
model name	: Intel(R) Xeon(R) Platinum 8259CL CPU @ 2.50GHz
cpu MHz		: 2499.998
```

## -A -B -C 显示匹配行的前后行

```bash
cat txt.txt
line1
line2
line3
line4
line5

#匹配内容和后一行 After
grep 'line3' -A 1
line3
line4

#匹配内容和前一行 Before
grep 'line3' -B 1
line2
line3

#匹配内容和前后一行 Combine
grep 'line3' -C 1
line2
line3
line4
```

# sed

## 常规替换

```bash
# -i 替换文件
sed -i 's/orgtext/replacetext/g' file
# 注意mac下，需要指定备份文件，这里''是不备份
sed -i '' 's/orgtext/replacetext/g' file
# 备份 .bak 文件
sed -i '.bak' 's/orgtext/replacetext/g' file
sed -i.bak 's/orgtext/replacetext/g' file
```

# xargs

## 逐行执行

```
# 把所有png文件改为jpg
ls *.png | xargs -I {} mv {} {}.jpg
```

## 批量去掉文件名前缀

```
# 支持前缀带空格
file_prefix="多余的前缀 " && find ./ -name "${file_prefix}*" | awk -F "${file_prefix}" '{print $2}' | xargs -I {} mv "${file_prefix}{}" "{}"
```

## 批量替换文件名中的关键字

```
filename_replace="纯享精选" && find . -name "*${filename_replace}*" | awk -F "${filename_replace}" '{printf "mv \"%s'"${filename_replace}"'%s\" \"%s%s\"\n",$1,$2,$1,$2}'
# 这里只是打印执行命令，可以复制在bash中再运行
```
